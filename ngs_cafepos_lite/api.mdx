# API & Integrations

<Info>
  API functionality and external integrations are currently unavailable in CafePOS Lite v1 but will be introduced with CafePOS Lite v2.
</Info>

<Card title="Coming in v2" icon="sparkles">
  CafePOS Lite v2 introduces support for multiple integration systems, payment handling, and service communication features.
</Card>

## Planned API Features

<CardGroup cols={2}>
  <Card title="Banking API" icon="building-columns">
    Automatic payment acceptance and decline support through Banking API integration.
  </Card>

  <Card title="VoCoVo Integrations" icon="headset">
    Integrated support for communication and service call systems.
  </Card>

## Supported Systems in v2

### Payment Integration

- Automatic accept & decline handling
- Banking API support

### Service Calls

- Applegate VoCoVo Go
- DRS VoCoVo Link
- DRS VoCoVo Go

### Planned Future Integrations

- Kybo Walvo
- easyPOS easyLink
